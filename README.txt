A couple simple Python script used for MLB The Shows Web App

TheShowScraper.py opens up pages for certain cards based args given

Usage:
python3 theshowscraper.py <rarity> <page number> 

PriceAlert.py sends a text message to you when a card reaches a certain price
The process every X minutes to check the price.
Message.py is ommitted which is the module that actually sends the text message
You'll need to set this up yourself https://www.twilio.com/docs/sms/quickstart/python
